# The Duck Lab: 오리 연구소

리액트(React)를 활용하여 구축한 웹 기반 심리테스트 플랫폼입니다.
직장인 유형 테스트를 시작으로, 다양한 주제의 테스트를 쉽게 확장할 수 있도록 컴포넌트 기반으로 설계되었습니다.

## 🔗 배포 링크 (Demo)
https://나쨩아이디.github.io/the-duck/

## 🛠 사용 기술 (Tech Stack)
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Library**: React 18, ReactDOM, Babel (CDN 방식)
- **Styling**: CSS (공통 모듈 분리 및 반응형 웹 적용)
- **Deployment**: GitHub Pages

## 📂 프로젝트 구조 (Directory Structure)
## ✨ 주요 기능 및 특징
1. **React 기반 SPA 아키텍처**
   - `useState`를 활용한 상태 관리로 페이지 새로고침 없는 부드러운 화면 전환 구현.
   - 컴포넌트 단위 개발로 코드 재사용성 및 유지보수 효율성 증대.

2. **확장성을 고려한 설계**
   - 메인 화면(`index.html`)과 개별 테스트(`office.html`)를 모듈화하여 새로운 테스트 추가 용이.
   - 공통 CSS 파일(`style.css`)을 통한 일관된 브랜드 디자인 시스템 구축.

3. **반응형 웹 디자인 (Responsive Design)**
   - PC 및 모바일 환경에 최적화된 UI/UX 제공.
   - Pretendard 폰트 적용으로 가독성 및 심미성 강화.
